API · Tools · list_tables
TOOL list_tables

List database tables

Returns every table the authenticated principal can read, with row counts and a one-line description from the schema's COMMENT. Backed by the sql://schema resource, this is the tool an agent calls first when it needs to ground a query in real data.

Arguments

schema*
Schema name to introspect. Defaults to the principal's primary schema. Wildcards accepted: analytics_*.
string
include_views
When true, materialized views and standard views appear alongside base tables.
boolean · false
min_rows
Filter out tables with fewer rows than this. Useful for pruning fixtures and migration tables out of agent context.
integer · 0
limit
Cap the returned list. Max 500.
integer · 100

MCP CLI
python · mcp-client⎘
# pip install mcp-client
from mcp import Client

async with Client(
    transport="sse",
    endpoint="https://mcp.linden.io",
    api_key=os.environ["LINDEN_KEY"],
) as mcp:
    res = await mcp.call(
        "list_tables",
        schema="analytics",
        min_rows=10,
        include_views=True,
    )

for t in res.tables:
    print(t.name, t.row_count)
